    340B Optimizer Features - Split Billing

    The 340B OPTIMIZER is a fully integrated solution in the industry. All necessary features reside on “one” web based platform.

    • Distributor Order and Contract Compliance Tools
    • Virtual Inventory Module
    • Formulary Management Tools
    Cost Containment / Seamless Ordering Process

    The ordering process works seamlessly with 340B OPTIMIZER. The auto-split function ‘splits’ purchases to the proper account based on real time accruals. This process is automated and occurs behind the scenes. The buyer can evaluate all WAC purchases BEFORE the order auto-splits, allowing edits to QTY purchased and switching to available alternates, all BEFORE the send and auto-split occurs.

    Evaluation Tools

    Features of 340B OPTIMIZER include the ability to view available alternate items with accumulations and price comparisons between products. These tools make formulary decisions easier.

    User-Friendly Software

    340B OPTIMIZER makes it easy to maintain a crosswalk, linking charge codes to NDC’s, view outpatient and inpatient accumulator buckets, edit billing units to coincide with Hospital billing system, and review all dispensed doses. All of this is available from ONE SCREEN.

    Audit Friendly

    Complete audit trail accountability is maintained through extensive tracking and reporting features that record all transactions. Reports can be generated by patient ID, ndc, wholesaler id number and more (audits are patient driven) making it easy to run reports.

    Hospital ID Number Integration

    Provides simple cross-walk management within the 340B OPTIMIZER. Hospital ID number integration allows a facility to keep an updated cross-walk file and easily identify the item being used.

    Flexibility

    Data can be NDC based or CDM based or both.

    Accurate Savings Report

    Each night after invoices are processed, an accurate Savings Report will be available. Savings Reports provide a comparison between the actual price paid on the purchase date to the GPO price that same day. 340B purchase savings and WAC purchase lost savings are taken into consideration which gives a complete picture of total overall savings.
